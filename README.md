This is a Full Stack Session Booking Application that allows users to book sessions through a user-friendly interface. The system includes a React frontend and a Spring Boot backend, enabling seamless data handling and user interaction.
The application is designed with a clean architecture and follows modern development practices suitable for real-world deployment.

Features
 Book sessions easily through a form
 Real-time data submission to backend
 REST API integration
 Clean UI with React
 Modular code structure
 Error handling and validation


 Tech Stack
Frontend:
ReactJS
HTML5
CSS3
JavaScript
Axios
Backend:
Spring Boot
Java
REST APIs
Database:
MySQL

Prerequisites

Make sure you have installed:

Node.js (v16+ recommended)
Java (JDK 17 recommended)
Maven
MySQL


How to Run the Project
1. Clone the Repository
git clone <your-repo-link>
cd <project-folder>

2. Run Backend (Spring Boot)
cd backend
mvn spring-boot:run

3. Run Frontend (React)
cd frontend
npm install
npm start

Common Issues & Fixes

❌ Axios not defined error
👉 Run:
npm install axios
